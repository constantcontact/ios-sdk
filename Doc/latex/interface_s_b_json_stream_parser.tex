\subsection{Detailed Description}
Parse a stream of J\-S\-O\-N data. 

Using this class directly you can reduce the apparent latency for each download/parse cycle of documents over a slow connection. You can start parsing {\itshape and return chunks of the parsed document} before the entire document is downloaded.

Using this class is also useful to parse huge documents on disk bit by bit so you don't have to keep them all in memory.

Expect multiple documents separated by whitespace. 

Normally the {\ttfamily -\/parse}\-: method returns S\-B\-Json\-Stream\-Parser\-Complete when it's found a complete J\-S\-O\-N document. Attempting to parse any more data at that point is considered an error. (\char`\"{}\-Garbage after J\-S\-O\-N\char`\"{}.)
